# E-Commerce Full-Stack Application

e-commerce application built with **React**, **Node.js**, **MongoDB**, **Next.js**, and **TypeScript**.

### Frontend
- **Next.js 14**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Zustand**
- **React Query**
- **Axios**

### Backend
- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **TypeScript**
- **JWT**
- **bcryptjs**
